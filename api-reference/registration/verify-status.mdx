---
title: "GET /api/verify/status"
description: "Returns the current verification state for an email or phone identifier on a specific channel, including attempt counts and time-to-live."
---

Returns the verification state for an identifier and channel.

## Query parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `identifier` | `string` | Yes | Email address or phone number |
| `channel` | `string` | Yes | `"email"` or `"sms"` |

## Response (200)

```json
{
  "identifier": "alice@example.com",
  "channel": "email",
  "verified": true,
  "verifiedAt": "2024-01-01T00:00:00Z"
}
```

`verifiedAt` is `null` when `verified` is `false`.
