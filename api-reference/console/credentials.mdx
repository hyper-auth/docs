---
title: "Console: Credentials"
description: "List and revoke an end user's passkeys and OAuth credentials. Credentials are created by sign-in flows and immutable from the console."
---

Inspect and revoke end-user credentials (passkeys, OAuth links). Credentials are created exclusively by the WebAuthn and OAuth sign-in flows — the console exposes only read and delete. Raw public keys are never returned.

## List credentials for an end user

```http
GET /v1/console/orgs/:orgId/apps/:appId/end-users/:userId/credentials
```

Required capability: `console.credential.read`.

### Query parameters

| Field | Type | Default | Description |
|---|---|---|---|
| `page` | integer | `1` | 1-indexed page number |
| `pageSize` | integer | `25` | Max `100` |

### Response (200)

```json
{
  "rows": [
    {
      "id": "cr_01HZ...",
      "endUserId": "eu_01HZ...",
      "type": "webauthn",
      "provider": "platform",
      "label": "iPhone passkey",
      "createdAt": "2026-05-19T10:00:00.000Z",
      "lastUsedAt": "2026-05-19T11:32:08.000Z"
    }
  ],
  "total": 1
}
```

The `publicKey` field is intentionally omitted from all responses.

## Get a single credential

```http
GET /v1/console/orgs/:orgId/apps/:appId/credentials/:id
```

Required capability: `console.credential.read`.

Returns `404 { "error": "NOT_FOUND" }` if the id does not exist.

## Revoke a credential

```http
DELETE /v1/console/orgs/:orgId/apps/:appId/credentials/:id
```

Required capability: `console.credential.delete`.

Hard-deletes the credential row. The end user can no longer authenticate with it. Returns `204 No Content`.

### Example

```bash
curl -X DELETE https://api.hyperauth.dev/v1/console/orgs/$ORG/apps/$APP/credentials/$CRED_ID \
  -H "Authorization: Bearer $UCAN_TOKEN"
```
